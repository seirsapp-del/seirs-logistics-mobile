Suggested Improvements (Industry Standard)

📦 Customer App

  • Live map on Home screenShow a live map with nearby drivers like Uber/Bolt; reduces friction to first booking
  • Scheduled delivery remindersPush notification 30 min before scheduled pickup time
  • Real-time ETA countdownLive ticking counter instead of static "20 mins away" on trip progress
  • Package insurance upsellOptional insurance add-on during booking (DHL/GIG standard)
  • Multiple recipientsSingle booking, different people receiving at each stop
  • Order status deep linksShareable live tracking URL so recipient can watch in browser without the app

🛵 Driver App

  • Earnings goal tracker"₦2,000 away from daily goal" — gamified motivation for more trips
  • Surge / demand heatmapShow high-demand zones on mini map so drivers position themselves better
  • Offline mode job queueCache accepted jobs locally so poor-signal areas don't break mid-delivery
  • One-tap navigationDeep-link into Google Maps / Waze with pre-filled destination; no manual copy
  • Weekly earnings summaryMonday morning push: "Here's your last 7 days" — builds platform loyalty

🖥 Admin Dashboard

  • Real-time ops mapAll active drivers and in-progress deliveries on a live map for ops visibility
  • Bulk reassignSelect multiple stuck deliveries and reassign to available drivers at once
  • Automated payout approvalsAuto-approve withdrawals below a threshold (e.g. < ₦50,000)
  • Driver supply vs. demand chartBy hour and zone — predicts and prevents driver shortfalls before they happen
  • Customer churn alertsFlag customers inactive for 30+ days for automated re-engagement campaigns

⚙️ Backend

  • Per-user rate limitingCurrently global throttle; per-user limits prevent individual API abuse
  • Cancellation grace window2-minute window after booking before cancellation fee applies
  • Webhook retry queueExponential back-off retries if Flutterwave webhook fails to deliver
  • Driver location historyStore last N GPS points per trip for route replay and dispute resolution
  • Delivery SLA alertsAlert ops when a delivery is in "assigned" state for more than 10 minutes with no pickup
