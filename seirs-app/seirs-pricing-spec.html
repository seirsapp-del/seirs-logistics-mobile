③ Dwell Times — how long the driver stays at each stop

Dwell time = the minutes the driver isn't driving. They're loading, waiting for the recipient, getting a signature. Nigerians don't move at developed-country speed, so we built a +2 min cultural buffer into every stop.

Formula

dwell_per_stop_minutes = setup_time(by category) + weight_tier_load(by kg) + vehicle_factor(by vehicle) + nigeria_cultural_buffer ← always added

Weight tiers (added on top of category setup)

Weight rangeExtra minutesWhy
0 – 5 kg+1Hand-carry, single trip
5 – 20 kg+3One trip, two-handed or hand-cart
20 – 50 kg+6Multiple trips or two-person carry
50 – 200 kg+12Trolley, multi-person, partial unload
200 – 500 kg+20Crew effort, full truck bed
500 kg ++40Full loading crew, possibly hydraulic lift

Nigeria cultural buffer

ComponentMinutes addedWhy
Baseline buffer (every stop)2Network for OTP/signature, generator noise, gate security, "wait let me call my brother"
Compound / estate stops+2Security log book, parking instructions, longer walk to door
Market / open stall+3Crowds, parking outside, haggling
Government building / bank+5ID check at gate, sign-in, escort to recipient

⑤ Fuel Model — pass-through, not absorbed

Nigerian fuel prices swing weekly. We don't hardcode km rates — we split each rate into labour (admin-set, stable) and fuel (computed from today's pump price ÷ vehicle efficiency). When petrol jumps, admin updates one number and every booking adjusts the same day.

Today's pump prices (admin updates these as needed)

FuelCurrent price (₦/litre)Last updated by admin
Petrol (PMS)950May 2026 — fill in real price
Diesel (AGO)1250May 2026 — fill in real price

Formula

fuel_cost_per_km = pump_price ÷ vehicle_efficiency displayed_km_rate = labour_per_km + fuel_cost_per_km # Worked: motorcycle on petrol # Petrol ₦950/L, motorcycle 45 km/L → fuel cost = ₦21/km # Labour (admin sets) = ₦100/km # Displayed rate = ₦121/km # Driver gets the FULL fuel cost back (so they break even on petrol) # Plus their labour share (separate, set in driver rate card)

⑦ Pricing Formula — universal

Same formula for every booking — ride, send, multi-stop, recurring, partner-store. Only the line items that apply get populated.

GROSS_PRICE = BASE_FARE + km × (LABOUR_PER_KM + FUEL_PER_KM) + stops × STOP_BONUS + estimated_dwell_minutes × DWELL_RATE ← only if > free threshold + CATEGORY_SURCHARGE % (fragile, frozen, animals…) + TIME_SURCHARGE % (night, peak) + ZONE_SURCHARGE % (inter-state, long-distance) − DISCOUNTS (loyalty, recurring, bulk) + VAT (7.5%) DRIVER_EARNS = DRIVER_BASE + km × (DRIVER_LABOUR_PER_KM + FUEL_PER_KM) ← driver gets fuel back + stops × DRIVER_STOP_BONUS + dwell_min × DRIVER_DWELL_RATE + 100% of TIPS + admin-set share of TIME & ZONE surcharges − fuel never deducted (driver paid it out of pocket, so reimbursed in full) PARTNER_STORE_EARNS (only for partner-store flows) = ₦500 per package handled + 70% of any overstay storage fees SEIRS_NET = GROSS_PRICE − DRIVER_EARNS − PARTNER_CUT − VAT_REMITTED

Three ways a business can create one

  1. Download SEIRS template — pre-built CSV with headers + example rows + comments. Open in Excel / Google Sheets, fill, upload. Lowest skill barrier.
  2. Export from their own system — map their columns to ours once (e.g. "Customer Name" → recipient_name). Mapping saved for future uploads.
  3. Paste from clipboard — for one-off bulk runs from a spreadsheet. Just paste rows in the upload screen.

Validation + preview flow

  1. Business uploads file
  2. Backend parses + validates each row (weight, category enum, phone format)
  3. Backend geocodes each address
  4. Backend computes per-row price + total
  5. Preview table shown: ✓ valid rows in green, ✗ flagged rows in red with reason
  6. Business fixes flagged rows in-app (or re-uploads corrected CSV)
  7. Confirms → wallet debits → dispatch starts

Common error messages business will see:

  • "Row 3: Address not found — please use a landmark or street name."
  • "Row 5: Weight 50 kg with motorcycle is unsafe. Choose van/truck or remove override."
  • "Row 8: Category 'pizza' not in catalogue. Try 'food_hot' instead."
  • "Row 12: Phone '0801234' invalid — Nigerian numbers need 11 digits."

⑭ Admin Controls — how the rate card stays current

Every number above is admin-editable in the dashboard. No engineers needed for inflation tuning or fuel-price spikes.

CapabilityBehaviour
Edit any rateInline form per row. Saving stamps the change with who, when, why.
"What is this for?" tooltipEvery editable field has a description so non-engineer staff understand what they're changing.
Inflation shortcut"Bump all labour rates by X%" button with confirm + audit log entry.
Fuel price updateOne field per fuel type. Updates auto-propagate to all displayed km rates.
