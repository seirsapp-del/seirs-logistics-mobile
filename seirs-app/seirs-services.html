Flutterwave (1.4% of GMV) is not in the total above because it scales with your revenue, not your costs. At 100K users, if average order value is ₦2,500 (~$1.50) and each active user places 4 deliveries/month: 20,000 active users × 4 × ₦2,500 = ₦200M GMV → 1.4% = ₦2.8M (~$1,700) in processing fees per month. At that scale, negotiate a custom rate — Flutterwave offers volume discounts at this level.
Google Maps at 100K users is the single largest cost driver. Mitigation strategies built into the codebase: debounce address autocomplete (don't call per keystroke), cache direction results for identical routes, use Geocoding only once per address, and batch driver location updates via WebSocket rather than per-update API calls. With aggressive caching, the $1,800 estimate could be reduced to $400–600.

Implement Account Lockout After Failed Login Attempts Security
Currently we throttle at 10 req/min per IP, but a determined attacker rotating IPs could still brute-force a weak password. Account lockout closes that gap — it's user-specific, not IP-specific.

    • On successful login: reset counter to 0
  2. No downtime required. TypeORM synchronize will add the columns automatically on next deploy.
